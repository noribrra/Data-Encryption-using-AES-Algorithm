Project Title: Data Encryption using AES Algorithm

Overview:
The project focuses on implementing data encryption using the Advanced Encryption Standard (AES) algorithm. It involves creating a software library in C# to encrypt texts and files, as well as developing a WinForm application to utilize this library for encryption tasks.

Designing and implementing the AESAlgorithm class
Functions for encrypting and decrypting byte arrays, text, and files
Summary of Chapter 2
Chapter 3: Developing the WinForm Application

Interface design for encryption operations
Programming button events for text and file encryption and decryption
Implementing file path selection functionality
Complete program code
References
Introduction:
The history of encryption dates back thousands of years, with encryption techniques used to secure sensitive communications. The project's importance lies in the ability of encryption to protect data from unauthorized access, tampering, and interception, making it a crucial information security tool.

Project Objectives:

Understand the fundamental concepts and working principles of the AES algorithm.
Develop a DLL software library using AES for text and file encryption in C#.
Utilize the DLL library in a WinForm application for encryption tasks.
Conclusion:
The project aims to contribute to information security by implementing robust encryption techniques using the AES algorithm, ensuring the confidentiality and integrity of data during transmission and storage.
